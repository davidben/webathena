ctlfish

Changing the volume on sipbmp3 has never been this fun!

This is a demonstration of Webathena's GSS-API implementation. It is a remctl client that runs entirely in the browser. A WebSockets to TCP proxy is used to communicate with the target server. (If WebSockets is unavailable, Socket.IO is used as fallback.) This page receives only credentials for the target service and those credentials themselves never leave your browser.

Run your own remctl server? Email davidben at MIT to get it added to the list. (The proxy needs a server whitelist so people can't use it to abuse anything that treats the MIT network as special.) Actually, I probably don't need a whitelist anymore now that it's running on this OpenShift thing. Will need to tweak that.
